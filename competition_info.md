# AFAC2026挑战组-赛题三：稀疏反馈下的自动化实验挑战

## 1. 赛事基本信息
- **赛事名称**：AFAC2026挑战组-赛题三：稀疏反馈下的自动化实验挑战
- **所属专题**：AFAC2026金融智能创新大赛
- **主办方**：蚂蚁集团等
- **奖金池**：¥ 1,000,000
- **标签**：#金融 #智能体 #蚂蚁集团竞赛

## 2. 任务背景
在真实科研与模型研发过程中，实验优化是一个持续的闭环（提出配置—运行试验—读取反馈—决定下一步）。本赛题聚焦于 **Agent 长程推理与自动化实验控制**，核心问题是在有限预算、稀疏反馈、禁止并行的约束下，构建能够持续修正实验策略的自动化实验智能体。

## 3. 任务目标
设计一个自动化实验 Agent 系统，能够自主完成以下流程：
1. **读取历史实验结果**：从先前的尝试中获取数据。
2. **分析反馈信号**：理解验证集指标、耗时、资源占用或失败日志。
3. **决定下一轮实验配置**：包括模型选择、超参数调整、训练策略等。
4. **持续迭代优化**：在预算耗尽前不断提升结果。
5. **输出最终预测结果**：提交最优方案的预测。

## 4. 子任务说明
赛题采用金融场景下的图学习任务作为测试：
- **子任务（1）产品分类任务**：图节点分类。在连接关系稀疏的节点上进行准确分类。
- **子任务（2）产品推荐任务**：图链接预测。为交互历史较少的新用户预测产品列表（按置信度排序）。

## 5. 核心能力要求
参赛 Agent 需要具备：
- **实验记忆维护**：在多轮交互中保留有效经验。
- **规律提炼**：从稀疏/有限反馈中总结规律并更新策略。
- **决策平衡**：在预算约束下进行合理的探索（Exploration）、利用（Exploitation）与停止决策。
- **经验迁移**：从不同数据集的实验中总结可迁移的经验。

## 6. 约束条件
- **有限预算**：实验次数或资源受限。
- **禁止并行**：必须串行进行实验决策。
- **稀疏反馈**：反馈信号有限，非完整监督。

## 7. 评估标准
不仅关注最终预测结果的准确性（分类准确率或推荐指标），还关注 Agent 系统在决策过程中的优化效率和策略组织能力。
